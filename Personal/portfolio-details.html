Vehicles Rental

Project informations

  • Category: Web Application
  • Project date: 2017
  • Project URL: Click here

This project consists of designing and developing a website for an online car rental agency. We want to offer a complete online car rental solution that offers the standard operating functions of a car rental company, namely searching for a car, reserving a car, returning a car, issuing its delivery note. This website has a graphical web interface, and it must also be able to connect to a database.
